* ubuntu_16.04搭建Qt开发环境

** 安装依赖

#+begin_src bash
sudo apt install -y gcc g++ make libgl1-mesa-dev
#+end_src

*** Ubuntu 22.04可能需要安装的依赖：
#+begin_src bash
sudo apt install -y gcc-12
#+end_src

*** 安装向日葵可能需要的依赖

#+begin_src bash
sudo apt --fix-broken install -y
#+end_src

*** 向日葵远程控制Ubuntu 22可能会遇到黑屏问题

安装并选择lightdm，然后重启操作系统
#+begin_src bash
sudo apt install -y lightdm
#+end_src

如果在上面选择默认的视频管理插件手滑，点错了。我们可以输入如下命令，重新配置默认视频管理插件。

#+begin_src bash
sudo dpkg-reconfigure lightdm
#+end_src

** 安装Qt

前提：已在官网注册账户

运行后，默认安装在家目录，如果需要的话也可以指定安装目录，我是默认安装在家目录，组件全选
#+begin_src bash
./qt-opensource-linux-x64-5.14.2.run
#+end_src
