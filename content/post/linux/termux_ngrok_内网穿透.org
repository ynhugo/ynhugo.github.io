* termux ngrok 内网穿透

** 前言
这篇教程以termux为例，当然其他系统也可以使用以下方法。

** 一、安装ngrok工具

安装ngrok工具建议去官网下载

[[https://ngrok.com/][ngrok官网链接]]

termux先在在命令行中输入uname -m确认内核

#+begin_src bash
  ~ $ uname -m
  aarch64
#+end_src

arrch64也是ARM64，所以在官网中获取下载链接后输入到命令行中

#+begin_src bash
  wget https://bin.equinox.io/c/4VmDzA7iaHb/ngrok-stable-linux-arm64.tgz
#+end_src

解压，给予执行权限并移动到$PREFIX/bin文件夹下

#+begin_src bash
  tar ngrok-stable-linux-arm64.tgz
  chomod +x ngrok
  mv ngrok $PREFIX/bin/
#+end_src

测试

** 解决遇到过的报错
- 启动服务后到127.0.0.1:4040点击了网页链接却报错：\\
  解决：端口要有服务在运行，比如在80端口配置了apache服务，命令就应该是

#+begin_src bash
  ~ $ ngrok http 80
  # 或
  ~ $ ngrok https 80
#+end_src

- 启动ngrok失败的可能性1\\
  没有安装两个软件

#+begin_src bash
  $ pkg install proot resolv-conf
#+end_src

- 启动ngrok失败的可能性2\\
  没有进入termux-chroot

#+begin_src bash
  $ termux-chroot
#+end_src
