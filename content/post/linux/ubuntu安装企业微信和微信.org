* 搭建环境

本人工作的计算机电脑系统：
Ubuntu18.04.3

* 下载Wine环境包

Wine环境包下载地址： http://archive.ubuntukylin.com/software/pool/partner/ukylin-wine_70.6.3.25_amd64.deb

** 什么是wine？

总结：Wine 是一款在 Linux 和 macOS 上能够运行 Windows 应用程序的兼容层软件，它可以使得用户能够在非 Windows 系统中运行 Windows 程序，从而方便用户的跨平台操作。

* 下载企业微信（wine）包

企业微信（wine）包下载地址： http://archive.ubuntukylin.com/software/pool/partner/ukylin-wxwork_1.0_amd64.deb

* 下载微信（wine）包

微信（wine）包下载地址： https://archive.ubuntukylin.com/software/pool/partner/ukylin-wechat_3.0.0_amd64.deb

* 安装

打开终端，进入下载文件所在目录，输入命令。

#+begin_src bash
  sudo apt-get install -f -y ./ukylin-wine_70.6.3.25_amd64.deb

  sudo apt-get install -f -y ./ukylin-wxwork_1.0_amd64.deb

  sudo apt-get install -f -y ./ukylin-wechat_3.0.0_amd64.deb
#+end_src
