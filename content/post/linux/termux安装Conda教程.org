* termux安装Conda教程

** 安装Ubuntu

#+begin_src bash
  #- 安装proot-distro
  pkg install proot-distro -y
  #- 查看有哪些系统
  proot-distro list
  #- 安装ubuntu系统
  proot-distro install ubuntu
  #- 登录ubuntu系统
  proot-distro login ubuntu
#+end_src

** 安装Conda

#+begin_src bash
  #- 下载miniconda安装脚本
  wget -v https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-aarch64.sh
  #- 运行安装脚本
  bash Miniconda3-latest-Linux-aarch64.sh
#+end_src
