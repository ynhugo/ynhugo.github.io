** hugo.yaml添加Giscus（GitHub用户首选）

*优势：*

✅ 基于GitHub Discussions
✅ 支持反应表情和回复
✅ 完全免费，无广告
✅ 支持多种主题

*配置步骤：*

1. 设置GitHub仓库：
   - 仓库必须是公开的
   - 安装 giscus app
   - 在仓库中启用Discussions功能
2. 配置hugo.yaml：

#+begin_src yaml
params:
    comments:
        enabled: true
        provider: giscus
        giscus:
            repo: username/repo-name
            repoID: R_kgDOxxxxxx
            category: General
            categoryID: DIC_kwDOxxxxxx
            mapping: pathname
            lightTheme: light
            darkTheme: dark
            reactionsEnabled: 1
            emitMetadata: 0
#+end_src
# 该配置在hugo.yaml中已经有模板了，把需要替换的信息替换一下就可以了
2. 或根据stach-starter模板配置config/_default/params.toml

#+begin_src yaml
## Comments
[comments]
    enabled  = true
    provider = "giscus"


    [comments.giscus]
        repo = "ynhugo/ynhugo.github.io"
        repoID = "R_xxxxxxxx"
        category = "Announcements"
        categoryID = "DIC_xxxxxxxxx"
        mapping = "pathname"
        lightTheme = "light"
        darkTheme = "dark"
        reactionsEnabled = 1
        emitMetadata = 0
#+end_src

3. 获取配置信息：
   - 访问 giscus.app
   - 输入仓库信息，获取配置代码
