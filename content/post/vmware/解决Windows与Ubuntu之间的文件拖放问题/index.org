* 方法一

打开终端，依次执行以下命令

#+begin_src bash
sudo apt-get autoremove open-vm-tools
sudo apt-get install open-vm-tools-desktop
# 或
sudo apt-get autoremove open-vm-tools；sudo apt-get install open-vm-tools-desktop
#+end_src

然后再重启Ubuntu即可。
