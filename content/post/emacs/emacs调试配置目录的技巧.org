* emacs调试配置目录的技巧

** 方法1（推荐）

#+begin_src bash
temp="$HOME/temp"
mkdir -p $temp
git clone https://gitee.com/emacsd/emacs.d $temp/.emacs.d

# 指在emacs启动时有效，不会影响到真实家目录
HOME=$temp emacs
#+end_src

** 方法2

#+begin_src bash
temp="$HOME/temp"
mkdir -p $temp; cd $temp
git clone https://gitee.com/emacsd/emacs.d $temp/.emacs.d
export HOME=$temp

# 运行emacs加载克隆下来的配置
emacs

# 如果要恢复HOME变量的默认值
# 查看HOME变量的默认值可以运行export -p发现OLDPWD变量对应的默认HOME变量值
export -p
HOME=$OLDPWD
#+end_src
