* Linux安装并启动Resilio

** 下载链接

https://download-cdn.resilio.com/2.7.3.1381/Debian/resilio-sync_2.7.3.1381-1_amd64.deb

** 安装

#+begin_src bash :tangle no
sudo dpkg -i resilio-sync_2.7.3.1381-1_amd64.deb
#+end_src

** 配置

#+begin_src bash :tangle no
sudo usermod -aG ubt rslsync
sudo usermod -aG rslsync ubt
#+end_src

** 建立同步文件夹

#+begin_src bash :tangle no
mkdir synced_folder
#+end_src

** 启动resilio

#+begin_src bash :tangle no
sudo service resilio-sync start
#+end_src

** 在浏览器中打开Resilio管理界面
Linux 上的 Resilio Sync 默认并没有 GUI 界面，但是提供了基于 Web 的界面。在通过上面的步骤启动了 Resilio Sync 之后，我们就可以在浏览器中输入如下地址打开 Resilio Sync 的 Web 管理界面
   
#+begin_src bash :tangle no
https://127.0.0.1:8888/gui/
#+end_src

** 注意事项

如果上面的地址打不开可以尝试检查一下是否有程序占用了 8888 端口

然后重新启动Resilio并在浏览器访问

#+begin_src bash :tangle no
netstat -apn | grep 8888
fuser -k -n tcp 8888
sudo service resilio-sync start
#+end_src

** 下载授权文件

Linux版本下载下来是专业版的，需要授权文件进行授权，要不然用不了Resilio

下载链接1：https://www.123pan.com/s/ZAzA-6D8wh

下载链接2：[[resources/ResilioSyncPro.btskey.zip][ResilioSyncPro.btskey.zip]]

然后在web界面添加授权文件即可
