* soul自动闭麦

** 使用场景

soul开派对，如果是一晚上不关派对且麦克风没有闭麦的情况下，当到了早上的时候，闹钟会响，然后会影响到派对里的人，这个时候有一个可以自动把soul的麦克风关闭的脚本在指定时间点自动运行，避免闹钟透过麦克风影响到其他人。

** python脚本

运行环境：termux
脚本运行前提：需要配置好uiautomator2和adb无线调试，无线调试开启后就可以使用127.0.0.1这个ip了。

#+begin_src python
import sys
import uiautomator2 as u2

d = u2.connect('127.0.0.1')

# 首先检查当前是否是开麦状态（假设开麦状态下按钮可能有特定属性）
# 这里需要根据实际UI情况调整判断条件
if d(resourceId="cn.soulapp.android:id/ivMicButton", description="闭麦按钮").exists:
    d(resourceId="cn.soulapp.android:id/ivMicButton", description="闭麦按钮").click()
#+end_src

** crontab定时任务

每天早上7点40自动运行soul自动闭麦python脚本。

#+begin_src bash
40 07 * * * bash $(python ~/soul自动闭麦.py)
#+end_src
