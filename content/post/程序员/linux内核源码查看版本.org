* linux内核源码查看版本

#+begin_src bash
# 方法一
# 进入内核源码目录执行
make kernelversion
# 方法二
# 指定Makefile所在目录执行
make -C linux-xlnx-xilinx-v2018.3 kernelversion
# 或把-C选项放到最后也可以
make kernelversion -C linux-xlnx-xilinx-v2018.3
# 方法三
# 指定Makefile所在目录找到指定Makefile
make -C linux-xlnx-xilinx-v2018.3 -f Makefile kernelversion
#+end_src
