* 官方源码

** 体验教程

*** 克隆仓库

#+begin_src sh
  git clone https://github.com/2noise/ChatTTS
  cd ChatTTS
#+end_src

*** 安装依赖

**** 1. 直接安装

#+begin_src sh
  pip install --upgrade -r requirements.txt
#+end_src

**** 2. 使用 conda 安装

#+begin_src sh
  conda create -n chattts python=3.12
  conda activate chattts
  pip install -r requirements.txt
#+end_src

*** 快速启动

确保在执行以下命令时，处于项目根目录下。

**** 1. WebUI 可视化界面

#+begin_src sh
  python examples/web/webui.py
#+end_src
