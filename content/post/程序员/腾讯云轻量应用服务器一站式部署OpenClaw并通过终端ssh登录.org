* 腾讯云轻量应用服务器一站式部署OpenClaw并通过终端ssh登录

** 轻量应用服务器 4核4G3M
活动价79元/月

选择镜像Moltbot

** 本地生成自定义名称的密钥对
#+begin_src bash
# 如果不想使用现有的id_rsa秘钥对，可以生成自定义秘钥对
ssh -t rsa -b 4096 -f ~/.ssh/moltbot

# 如果不想指定秘钥名称，默认id_rsa秘钥对使用最方便
ssh -t rsa -b 4096
#+end_src


** 配置SSH公钥
#+begin_src bash
# 查看公钥
cat ~/.ssh/moltbot
#+end_src
将公钥粘贴到服务器的~/.ssh/authorized_keys中

** 本地SSH登录
#+begin_src bash
# 如果是生成的自定义私钥对，需要指定私钥
ssh -i ~/.ssh/moltbot root@101.43.216.165

# 如果使用id_rsa秘钥对，可以不用指定私钥
ssh root@101.43.216.165
#+end_src
如果不想指定私钥，可以把~/.ssh/id_rsa.pub配置到服务器的authorized_keys中，这样ssh会默认使用私钥id_rsa登录

** 配置moltbot
#+begin_src bash
# 配置moltbot
clawdbot onboard
#+end_src

** 准备工作
#+begin_src bash
# 端口转发
ssh -N -L 18789:127.0.0.1:18789 moltbot
#+end_src
然后在本地访问18789端口即可，如：http://localhost:18789/

** 查看dashboard
#+begin_src bash
clawdbot dashboard
# 或
ssh openclaw "bash -l -c clawdbot dashboard"

openclaw dashboard
# 或
ssh openclaw "bash -l -c openclaw dashboard"
#+end_src
