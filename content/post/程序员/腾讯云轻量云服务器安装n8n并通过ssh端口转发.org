* 腾讯云轻量云服务器安装n8n并通过ssh端口转发

** 安装nodejs指定版本
#+begin_src bash
# 安装 nvm（如果尚未安装）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 重新加载 shell 配置
source ~/.bashrc

# 安装兼容的 Node.js 版本（建议使用 LTS 版本）
nvm install 20.19.0  # 或 nvm install 20（最新 20.x 版本）

# 切换到新安装的版本
nvm use 20.19.0

# 验证版本
node --version
#+end_src

** 安装n8n
#+begin_src bash
# 使用npm全局安装n8n
npm install n8n -g

# 安装完成后，通过以下方式启动 n8n：
n8n
# 或
n8n start
#+end_src

** 如果要更新
#+begin_src bash
# 要将你的 n8n 实例更新到最新版本 ，请运行：
npm update -g n8n

# 安装下一版本：
npm install -g n8n@next
#+end_src

** 端口转发
#+begin_src bash
# 启动n8n后会给出一链接，端口为5678，则本地执行如下命令将服务器端口转发到本地：
# ssh -N -L 5678:127.0.0.1:5678 root@192.168.x.x
#+end_src
