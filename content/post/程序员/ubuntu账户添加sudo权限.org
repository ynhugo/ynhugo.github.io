* ubuntu账户添加sudo权限

** 解决方案

方法1：使用已有sudo权限的账户
使用系统中已有的具有 sudo 权限的账户登录

然后执行：sudo adduser username sudo

方法2：直接修改sudoers文件
使用root账户登录（如果有）

编辑sudoers文件：visudo

添加一行：username ALL=(ALL:ALL) ALL

保存退出

方法3：单用户模式修改（需要物理访问）
重启系统，在GRUB菜单中选择恢复模式

进入root shell

执行：adduser username sudo

重启系统

方法4：使用su切换到root（如果知道root密码）
执行：su -

输入root密码

执行：adduser username sudo

** 注意事项
修改sudoers文件时要特别小心，错误的编辑可能导致系统无法使用sudo

建议优先使用方法1，即使用已有sudo权限的账户来添加新权限

如果没有其他sudo用户，可能需要联系系统管理员或使用物理访问权限来解决问题
