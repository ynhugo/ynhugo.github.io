* exFAT文件系统的支持
exFAT 是 Windows 常用的文件系统格式（常见于 U 盘、SD 卡和移动硬盘），但 Ubuntu 默认不包含 exFAT 驱动。

#+begin_src bash
# 安装 exFAT 工具包
sudo apt install -y exfat-fuse exfat-utils
#+end_src
安装完成后，重新插入设备。
