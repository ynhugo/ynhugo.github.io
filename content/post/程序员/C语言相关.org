* 详解 const uint8_t * const* 类型：

const uint8_t * const* 是 C 语言中的一种复杂指针类型，用于指向指向常量的指针。让我们详细拆解这个类型：

** 类型拆解

1. const uint8_t：
   - uint8_t 是无符号的 8 位整数类型，通常用于表示一个字节。
   - const 关键字在 uint8_t 前面，表示这个 uint8_t 类型的数据是常量，即在指针指向的数据位置上的值不能被修改。

2. const uint8_t *：
   - 这是一个指向 const uint8_t 类型的指针。
   - 它表示你可以通过这个指针访问 const uint8_t 类型的数据，但不能修改这些数据。

3. const uint8_t * const：
   - 这是一个指向 const uint8_t 的指针，但这个指针本身也是常量。
   - 也就是说，你不能通过这个指针去修改指针的值（即你不能改变指针所指向的位置），但你依然可以读取它所指向的数据。

4. const uint8_t * const*：
   - 这是一个指向 const uint8_t * const 的指针，也就是一个指向常量指针的指针。
   - 这意味着你有一个指针，它指向另一个指针，而这个被指向的指针是 const 的（即你不能修改它指向的位置），并且这个被指向的位置的数据也是 const 的（即你不能修改这些数据）。

** 用途和示例

这个类型经常用于表示一个指针数组，其中每个指针指向一个常量数据，并且这些指针本身也是不可修改的。它在处理图像数据时非常有用，因为图像数据通常是由多个平面组成的，每个平面的数据可能需要被读取但不应被修改。

*** 示例

假设我们有如下的数据：

#+begin_src c
  const uint8_t *plane1;  // 指向第一个平面的数据，数据不能被修改
  const uint8_t *plane2;  // 指向第二个平面的数据，数据不能被修改

  const uint8_t * const planes[] = { plane1, plane2 };  // 指针数组，数组中的每个指针都指向常量数据

  const uint8_t * const * planes_ptr = planes;  // 指向指针数组的指针，不能修改指针数组中的指针
#+end_src

在这个示例中：

- planes 是一个指针数组，其中每个元素都是指向常量数据的指针。
- planes_ptr 是一个指向这个指针数组的指针，你不能通过 planes_ptr 修改 planes 中的任何指针值，也不能通过这些指针修改数据本身。

** 具体应用

在 FFmpeg 的 sws_scale 函数中，const uint8_t * const* 用于指向图像的多个平面数据。每个平面都是不可修改的，这保证了图像数据在转换过程中不会被意外改变。同时，const uint8_t * const* 确保你不能改变指向这些平面的指针数组的结构。这种方式增加了代码的安全性和可维护性。

* C语言打印格式%x与%#x有什么区别

在 C 语言中，%x 和 %#x 是 printf 函数中的格式说明符，用于打印整数值的十六进制表示。它们之间的主要区别在于是否显示前缀：

1. %x:
   - 打印整数的十六进制表示，不带任何前缀。
   - 输出的是一个十六进制数，只使用小写字母（a 到 f）。
   - 示例：
     #+begin_src c
	   int num = 255;
	   printf("%x\n", num);  // 输出 "ff"
     #+end_src

2. %#x:
   - 在十六进制表示前加上 0x 前缀。
   - 也输出小写字母（a 到 f）。
   - 示例：
     #+begin_src c
	   int num = 255;
	   printf("%#x\n", num);  // 输出 "0xff"
     #+end_src

** 举例说明

#+begin_src c
  #include <stdio.h>

  int main() {
	  int num = 255;

	  // 使用 %x
	  printf("Using %%x: %x\n", num);  // 输出: Using %x: ff

	  // 使用 %#x
	  printf("Using %%#x: %#x\n", num);  // 输出: Using %#x: 0xff

	  return 0;
  }
#+end_src

** 总结

- "%x" 输出的是纯十六进制数值，不带前缀。
- "%#x" 在十六进制数值前加上 0x 前缀，标识这是一个十六进制数。

* 从终端输入字符串，提取字符串中的ASCII值 - ChatGPT

#+begin_src 
#include <stdio.h>
#include <stdlib.h>

int main(void) {
    unsigned char *arr = NULL;  // 初始化为 NULL
    unsigned int arr_num = 0;   // 当前输入字符的数量
    unsigned int capacity = 1;   // 初始容量设置为 1
    int ch;

    // 为 arr 分配初始内存
    arr = (unsigned char *)malloc(capacity * sizeof(unsigned char));

    printf("请输入包含任意大小数字的任意长度的字符串 (按 Enter 键结束):\n");

    while ((ch = getchar()) != '\n') {
        // 检查是否需要扩展数组
        if (arr_num >= capacity) {
            capacity *= 2; // 扩大容量为原来的两倍
            unsigned char *temp = realloc(arr, capacity * sizeof(unsigned char));
            arr = temp; // 更新 arr 指针
        }
        arr[arr_num++] = (unsigned char)ch; // 将读取的字符存储到数组
    }

    // 为字符串结束符增加空间
    unsigned char *final_arr = realloc(arr, (arr_num + 1) * sizeof(unsigned char));    
    arr = final_arr;
    arr[arr_num] = '\0'; // 添加字符串结束符

    // 输出读取到的字符串
    printf("您输入的字符串是: %s\n", arr);

    // 打印每个字符及其 ASCII 值
    for (unsigned int i = 0; i < arr_num; i++) {
        printf("arr[%d] = %c\n", i, arr[i]);
    }

    free(arr); // 释放动态分配的内存
    return 0;
}
#+end_src

** 代码说明
动态内存分配: 使用 malloc 为 arr 分配初始内存，之后根据需要使用 realloc 扩展内存。

字符读取: 使用 getchar() 从标准输入逐个读取字符，直到遇到换行符 \n。

容量管理: 每当字符数量达到当前容量时，容量翻倍，以合理利用内存。

字符串结束符: 在读取完成后，为字符串添加结束符 \0，确保可以正确作为字符串处理。

内存释放: 程序结尾处释放分配的内存，以避免内存泄漏。

*** 使用说明
编译并运行程序，输入一段字符并按回车键结束。
程序会输出您输入的字符串和每个字符的 ASCII 值。

* 提取字符串中的正数、浮点数 - ChatGPT

#+begin_src

// 提取数字的函数
void extractNumbers(const char *inputString) {
    double num = 0.0;
    int foundNumber = 0;  // 标志是否找到了数字
    int isNegative = 0;   // 标志是否为负数
    int hasDecimal = 0;   // 标志是否有小数部分

    while (*inputString) {
        // 跳过空格
        while (isspace(*inputString)) {
            inputString++;
        }

        // 检查正负号
        if (*inputString == '-') {
            isNegative = 1;
            inputString++;
        } else if (*inputString == '+') {
            inputString++; // 忽略正号
        }

        // 提取数字部分
        num = 0.0;
        hasDecimal = 0;
        double decimalPlace = 1.0;

        while (isdigit(*inputString) || *inputString == '.') {
            if (*inputString == '.') {
                hasDecimal = 1;
                inputString++;
                continue;
            }

            if (hasDecimal) {
                decimalPlace *= 10.0; // 更新小数位
                num = num + (*inputString - '0') / decimalPlace; // 加入小数部分
            } else {
                num = num * 10 + (*inputString - '0'); // 加入整数部分
            }
            inputString++;
        }

        // 应用符号
        if (isNegative) {
            num = -num;
            isNegative = 0; // 重置标志
        }

        // 判断并打印结果
        if (hasDecimal) {
            printf("提取的浮点数是: %.2f\n", num);
        } else {
            printf("提取的整数是: %d\n", (int)num);
        }

        foundNumber = 1;  // 找到数字

        // 重置状态
        while (isspace(*inputString)) {
            inputString++; // 跳过空格
        }
    }

    if (!foundNumber) {
        printf("没有找到任何数字。\n");
    }
}
#+end_src

** 代码说明

*** 主函数：
- 定义了输入字符串 inputString，并调用 extractNumbers 函数。

*** extractNumbers 函数：
- 使用 double 类型来存储提取的数值，以便处理浮点数。
- 使用 isNegative 标志来判断数字是否为负数。
- 使用 hasDecimal 标志来判断是否有小数部分。
- 遍历字符串时，首先跳过空格，然后检查正负号。
- 提取数字部分，包括整数和小数：
  - 如果遇到小数点，更新 hasDecimal 标志，并在后续循环中处理小数部分。
- 输出提取的数字，区分整数和浮点数。
