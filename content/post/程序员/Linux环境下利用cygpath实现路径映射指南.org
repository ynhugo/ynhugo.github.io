* Cygpath 命令的基本使用
Cygpath 命令的基本使用相对简单，它主要接受两个参数：第一个参数是 -w 或 -u，用于指定转换的方向（Windows 到 Unix 或 Unix 到 Windows），第二个参数是需要转换的路径。下面将介绍如何使用 Cygpath 来进行路径映射。

** 4.1 将 Windows 路径转换为 Unix 路径
如果你想将一个 Windows 路径转换为 Unix 路径，可以使用 -u 参数。例如：
#+begin_src bash
cygpath -u "C:\Program Files"
#+end_src

这将会输出对应的 Unix 风格的路径。

** 4.2 将 Unix 路径转换为 Windows 路径
如果你想将一个 Unix 路径转换为 Windows 路径，可以使用 -w 参数。例如：
#+begin_src bash
cygpath -w "/cygdrive/c/Program Files"
#+end_src

这将会输出对应的 Windows 风格的路径。

** 4.3 查看帮助信息
如果你需要查看 Cygpath 的帮助信息，可以使用 -h 参数或者不带任何参数直接调用 Cygpath 命令。
#+begin_src bash
cygpath -h
#+end_src

这将显示 Cygpath 的使用说明和可用的选项。
