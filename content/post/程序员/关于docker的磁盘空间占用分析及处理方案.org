*** 已执行操作

*** 当前可用空间

*** 后续建议

**** 如需进一步释放空间

1. 删除未使用的镜像:
   #+BEGIN_SRC bash
   docker rmi nginx:latest
   #+END_SRC
   - 释放: 240MB

2. 清理构建缓存
   #+BEGIN_SRC bash
     docker system prune
   #+END_SRC
   - 只清理构建缓存


3. 清理所有未使用资源:
   #+BEGIN_SRC bash
     docker system prune -a
   #+END_SRC
   - 注意: 会删除所有未使用的镜像(包括 petalinux)

**** 如需保留镜像但减少占用

- 镜像已压缩存储，无法进一步精简
- 可考虑使用多阶段构建减少最终镜像大小(需修改Dockerfile)
