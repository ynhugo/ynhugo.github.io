* 依赖wsl环境的docker运行容器需要调整区分大小写

#+begin_src bash
# 检查当前区分大小写
fsutil.exe file queryCaseSensitiveInfo <path>

# 修改区分大小写
# 若要更改 Windows 文件系统中的目录，使其区分大小写（FOO ≠ foo），请以管理员身份运行 PowerShell 并使用以下命令（如果cygwin下非管理员也可以）：
fsutil.exe file setCaseSensitiveInfo <path> enable
# 若要将 Windows 文件系统中的目录更改回不区分大小写的默认（FOO = foo），请以管理员身份运行 PowerShell 并使用以下命令（如果cygwin下非管理员也可以）：
fsutil.exe file setCaseSensitiveInfo <path> disable
#+end_src
