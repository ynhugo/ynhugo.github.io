* Windows安装Docker-Desktop到D盘

官网下载Docker Desktop Install.exe，官网链接：https://www.docker.com/products/docker-desktop/

** Docker Desktop指定安装到D盘

#+begin_src bash
.\"Docker Desktop Installer.exe" install --accept-license --installation-dir="D:\DockerDesktop\installation"
#+end_src
