* Android逆向入门教程

前提：安装Android Studio

#+begin_src bash
wget -c -e "https_proxy=127.0.0.1:10808" https://github.com/SuperMonster003/AutoJs6/releases/download/v6.6.4/autojs6-v6.6.4-arm64-v8a-110e6b64.apk
wget -c -e "https_proxy=127.0.0.1:10808" https://github.com/iBotPeaches/Apktool/releases/download/v2.12.1/apktool_2.12.1.jar
java -jar apktool_2.12.1.jar d autojs6-v6.6.4-arm64-v8a-110e6b64.apk -o tempdir
#+end_src
