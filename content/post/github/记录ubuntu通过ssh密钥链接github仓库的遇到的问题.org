记录ubuntu通过ssh密钥链接github仓库的遇到的问题

原本是正常的生成密钥，然后将密钥的公钥添加到github仓库中的

正常到这里就可以了，接下来只需要正常操作仓库就可以了，但是仓库拉取不下来分支

但是遇到了问题，遇到了终端打印"sign_and_send_pubkey: signing failed: agent refused operation"的问题

[[https://blog.csdn.net/someby/article/details/85329063][Ubuntu下解决sign_and_send_pubkey: signing failed: agent refused operation问题]]

解决方法，只需要如下操作即可

#+begin_src bash :tangle no
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/github_id_rsa
ssh -T git@github.com
#+end_src

然后重新去pull仓库就可以成功了
