* 脚本内容

创建一个txt文件如test.txt，将以下脚本内容添加到txt文件中，如果需要通过Vmware安装虚拟操作系统到VHD文件，那么先不分离vdisk，即先不执行 =detach vdisk= ,当然也可以在脚本执行完后手动手动再连接磁盘文件，即先执行 =select vdisk filed:\test_vhd.vhd= ，然后再执行 =attach vdisk= ，这个时候就可以在vmware中添加本地硬盘的时候看到对应磁盘文件的ID了

#+begin_src bash
create vdisk file=d:\test_vhd.vhd maximum=1024 type=fixed
select vdisk file=d:\test_vhd.vhd
attach vdisk
create partition primary
assign letter=Z
format quick label=vhd 
detach vdisk
#+end_src
在D盘根目录创建一个1G的固定磁盘，选中并连接该磁盘文件，接着创建主分区并快速格式化，然后分离该磁盘文件

** 脚本执行方式

#+begin_src bash
diskpart -s test.txt
#+end_src
