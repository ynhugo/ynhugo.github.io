使用qemu搭建arm嵌入式linux开发环境

使用说明

该脚本只有在第一次运行的时候需要构建环境，花费时间长一些，构建完成以后自动启动环境，以后再运行该脚本的时候就不需要再次构建环境了，加载后进入开发环境，敲回车确认。

退出时先按快捷键"Ctrl+x"，再按"x"即可退出环境。

	  echo "匹配行已存在，并且重启了nfs服务"

	  sudo su -c "cat >> /etc/exports << EOF
  /home/${USER} *(rw,nohide,insecure,no_subtree_check,async,no_root_squash)
  EOF"

	  echo "匹配行已添加，并且重启了nfs服务"

	  make modules -j12;

  fi

	  tar xvf $work_dir/$file_busybox -C $work_dir;
  fi
  if [ -d "$work_dir/$busybox/_install" ]; then
	  echo "_install exists."
  else
	  cd  $work_dir/$busybox;
	  make defconfig -j12;
	  make defconfig;
	  make -j12;
