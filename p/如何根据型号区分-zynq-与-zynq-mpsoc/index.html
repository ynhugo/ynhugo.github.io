如何根据型号区分 Zynq 与 Zynq MPSoC

一句话速查表

最直接的方法：看型号开头。

  • XC7Z → Zynq-7000（即通常说的 Zynq）
  • XCZU → Zynq UltraScale+ MPSoC（即 ZynqMP）

详细区分方法

1. 看前缀：一眼识别

前缀特征对应系列
XC7Z...Zynq-7000
XCZU...Zynq UltraScale+ MPSoC

> 特殊前缀说明： > - XA7Z... 或 XQ7Z... 仍为 Zynq-7000（车规/军工级） > - XAZU... 或 XQZU... 仍为 Zynq MPSoC

2. 核心差异速查表

对比项Zynq-7000Zynq MPSoC
处理器双核 ARM Cortex-A9四核 Cortex-A53 + 双核 Cortex-R5
工艺28nm16nm
内存支持DDR3DDR4
典型主频~1GHzA53: ~1.5GHz
常见型号XC7Z010, XC7Z020, XC7Z045XCZU2CG, XCZU3EG, XCZU7EG, XCZU19EG
高速外设PCIe Gen2, USB 2.0PCIe Gen3/4, USB 3.0, SATA, DP
典型应用工业控制、嵌入式视觉、ADASAI推理、4K/8K视频、云加速、SDR

型号示例

1
2

Zynq-7000:     XC7Z010-1CLG400C, XA7Z020-1CLG484Q
Zynq MPSoC:    XCZU3EG-1SFVC784E, XAZU4EV-1SFVC784Q

后缀速查（Zynq MPSoC）

后缀含义
CG仅 Cortex-A53（无 R5 和 GPU）
EGA53 + R5 + GPU（最常用）
EVEG + 视频编解码器（H.264/H.265）

总结

  • 记住一条核心规则：**7Z → 老款 Zynq，ZU → 新款 ZynqMP**
  • 选型建议：需要高算力、DDR4、多核异构、AI/视频 → 选 ZynqMP；成本敏感或已有成熟方案 → Zynq-7000
